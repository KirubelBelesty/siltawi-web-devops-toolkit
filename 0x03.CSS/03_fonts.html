The font property in CSS is used to specify the font of an element. The font property can be used to specify the font family, font size, font weight, font style, line height, letter spacing, and word spacing of an element. The font property can be specified as a single value, or as a list of values separated by commas.

About

The font property can also be used to specify a custom font. To do this, you need to use the @font-face rule. The @font-face rule allows you to load a custom font from a remote server or from a locally-installed font on the user's own computer.

Here are some of the most common values that can be used with the font property:
  • font-family: The font family is the name of the font that you want to use. For example, "Arial" or "Times New Roman".
  • font-size: The font size is the size of the font in pixels. For example, "16px" or "1.2em".
  • font-weight: The font weight is the thickness of the font. For example, "bold" or "normal".
  • font-style: The font style is the style of the font. For example, "italic" or "normal".
  • line-height: The line height is the spacing between lines of text. For example, "1.5" or "2em".
  • letter-spacing: The letter spacing is the spacing between letters. For example, "0.1em" or "normal".
  • word-spacing: The word spacing is the spacing between words. For example, "0.1em" or "normal".
